Abstract

Phys-GS Lite augments 3D Gaussian Splatting with semantic separation and physics-inspired dynamics. Using simple geometry heuristics, the method distinguishes rigid and deformable splats (e.g., ship vs water). A sinusoidal wind field introduces wave-like motion, while Laplacian smoothing ensures coherent deformations. This lightweight prototype demonstrates that physical priors can extend Gaussian Splatting toward dynamic, material-aware rendering without heavy neural training.
